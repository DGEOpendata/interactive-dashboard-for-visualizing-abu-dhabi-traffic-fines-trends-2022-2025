# Abu Dhabi Traffic Fines Dashboard (2022-2025)

## Overview
This interactive dashboard enables users to explore and analyze traffic fines data from Abu Dhabi for the years 2022 to 2025. It provides visual insights into ticket types, descriptions, and trends, helping users make data-driven decisions and understand traffic violation patterns.

## Features
- Filter traffic fines by year.
- Visualize fines distribution across ticket types and descriptions.
- Interactive bar charts for better data exploration.

## Prerequisites
- Python 3.7+
- Required libraries: pandas, plotly, dash

## Installation
1. Clone the repository:
   sh
   git clone https://github.com/your-repo/traffic-fines-dashboard.git
   
2. Navigate to the project directory:
   sh
   cd traffic-fines-dashboard
   
3. Install the required dependencies:
   sh
   pip install -r requirements.txt
   

## Usage
1. Place the dataset file (`Traffic_Fines-v3.0 (values) 2022-2025.xlsx`) in the project directory.
2. Run the application:
   sh
   python app.py
   
3. Open your browser and navigate to `http://127.0.0.1:8050/` to access the dashboard.
